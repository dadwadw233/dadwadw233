A high-fidelity navigation simulator for dynamic Gaussian Splatting scenes, connecting photorealistic rendering with embodied navigation training and evaluation.

### 02 / [BoxDreamer](https://github.com/zju3dv/BoxDreamer) · ICCV 2025

Generalizable 6-DoF object pose estimation from a single RGB image by predicting 3D bounding-box corners, without CAD models or category-specific training.

[Project page](https://zju3dv.github.io/boxdreamer/) · [Paper](https://arxiv.org/abs/2504.07955) · [Demo](https://huggingface.co/spaces/LittleFrog/BoxDreamer)
